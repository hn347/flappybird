High Level Design

About

Flappy Bird was chosen because its gameplay is simple yet demanding from a real-time systems perspective. Smooth animation, precise collision detection, and responsive controls require careful timing and efficient memory usage, especially when implemented without dedicated graphics hardware. The project builds on publicly available RP2040 VGA graphics libraries. Several characteristics of our code, like the use of direct digital synthesis or fixed point arithmetic, are adapted versions of the models provided by Bruce Land or Hunter Van Adams on the Cornell ECE 4760 website. All third-party code and references are explicitly cited in later sections of this report.
